# Shopmellostyles

## Deployment Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/mellostyles/Shopmellostyles.git
   cd Shopmellostyles
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the project:
   ```bash
   npm run build
   ```

## Setup Instructions
1. Ensure you have Node.js installed (version 14 or higher).
2. Configure environment variables as needed in a `.env` file.
3. Start the application:
   ```bash
   npm start
   ```
4. Access the application at `http://localhost:3000`

## Notes
- Ensure to have a database set up if applicable. Refer to the documentation for further details on configuration.
